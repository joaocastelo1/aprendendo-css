Aprenda CSS com Animações e Código

Veja a animação e o CSS usado, lado a lado, para aprender na prática!

1. Transição (Transition)

Muda suavemente propriedades como cor e transformação ao passar o mouse.

Hover
.transition-box {
  transition: transform 0.4s ease, background-color 0.4s ease;
}
.transition-box:hover {
  background-color: #f1c40f;
  transform: scale(1.2) rotate(10deg);
}

2. Transformação 3D (Transform)

Rotacione e escale elementos em 3D na interação.

3D Rotate
.transform-box {
  background: #e67e22;
  transition: transform 0.5s ease;
  perspective: 600px;
}
.transform-box:hover {
  transform: rotateY(180deg) rotateX(20deg) scale(1.1);
  background: #d35400;
}

3. Animação Pulsante (Keyframes)

Cria efeito de pulsação usando 'keyframes' em loop infinito.

Pulse
@keyframes pulse {
  0% {
    transform: scale(1);
    box-shadow: 0 0 6px #f39c12;
  }
  50% {
    transform: scale(1.15);
    box-shadow: 0 0 20px #f1c40f;
  }
  100% {
    transform: scale(1);
    box-shadow: 0 0 6px #f39c12;
  }
}
.pulse-box {
  background: #f39c12;
  animation: pulse 2.5s infinite ease-in-out;
}

4. Animação Slide + Fade

Elemento desliza da esquerda para a direita e some em loop.

Slide & Fade
@keyframes slidefade {
  0% {
    opacity: 0;
    transform: translateX(-50px);
  }
  50% {
    opacity: 1;
    transform: translateX(0);
  }
  100% {
    opacity: 0;
    transform: translateX(50px);
  }
}
.slidefade-box {
  background: #d35400;
  color: #fff;
  animation: slidefade 3s infinite ease-in-out;
}

5. Ciclo de Cores

Revolução de cores com brilho que se repetem infinitamente.

Color Cycle
@keyframes colorCycle {
  0% {
    background-color: #f39c12;
    box-shadow: 0 0 10px #f39c12;
  }
  25% {
    background-color: #e67e22;
    box-shadow: 0 0 20px #e67e22;
  }
  50% {
    background-color: #f1c40f;
    box-shadow: 0 0 15px #f1c40f;
  }
  75% {
    background-color: #d35400;
    box-shadow: 0 0 10px #d35400;
  }
  100% {
    background-color: #f39c12;
    box-shadow: 0 0 10px #f39c12;
  }
}
.colorcycle-box {
  animation: colorCycle 5s infinite linear;
}
