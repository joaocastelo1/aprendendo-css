🎨 Estilização com CSS (interno, inline e externo)

🖼️ Imagens e background

🅰️ Tipografia (fontes, tamanhos, cores)

📏 Unidades de medida (px, %, em, rem)

🔲 Bordas, margens e paddings

🧭 Flexbox

📦 Grid Layout (em breve)
